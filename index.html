Gear Packages

Package repositories for Gear.

Debian / Ubuntu (APT)

curl -fsSL https://clleb.github.io/gear-packages/apt/key.gpg \
  | sudo tee /etc/apt/keyrings/gear.gpg > /dev/null
echo "deb [signed-by=/etc/apt/keyrings/gear.gpg] https://clleb.github.io/gear-packages/apt stable main" \
  | sudo tee /etc/apt/sources.list.d/gear.list
sudo apt update && sudo apt install gear

Fedora / RHEL / openSUSE (RPM)

sudo dnf config-manager --add-repo https://clleb.github.io/gear-packages/rpm/gear.repo
sudo dnf install gear
