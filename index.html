Gear Packages

Package repositories for Gear — the AI-powered developer terminal.

Debian / Ubuntu (APT)

curl -fsSL https://portfolio.kyere.me/gear-packages/apt/key.gpg   | sudo tee /etc/apt/keyrings/gear.gpg > /dev/null
echo "deb [signed-by=/etc/apt/keyrings/gear.gpg] https://portfolio.kyere.me/gear-packages/apt stable main"   | sudo tee /etc/apt/sources.list.d/gear.list
sudo apt update && sudo apt install gear

Fedora / RHEL / openSUSE (RPM)

sudo dnf config-manager --add-repo https://portfolio.kyere.me/gear-packages/rpm/gear.repo
sudo dnf install gear
